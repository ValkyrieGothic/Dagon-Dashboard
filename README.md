# Dagon-Dashboard
A Dashboard for all your scripts. One click run any python script


Place script in your preferred directory. 

Run script

Script will create 3 folders in your directory

Apps - where you place your scripts
Logs - obvious, its for the app logs
Requirements - this is the file where YOUR scripts requirements will be held, if any.

Place your scripts in the Apps folder created. 

Run script again

Script will scan your scripts for requirements. If any are missing it will tell you.

Click on "Launch: of any script to open it immediately.
